COGS 127 // HUMAN-COMPUTER INTERACTION

Connecting Intuitive Design to Complex Interaction

Designing a responsive, accessible interface tailored for user engagement and seamless interactive workflows.

Role

UX/UI Designer & Researcher

Timeline

Spring 2026

Tools Used

Figma, HTML, CSS

Context

UC San Diego Course Project

01. The Challenge

In modern interactive ecosystems, users often experience friction when navigating data-heavy layouts. The objective of this project was to synthesize key human-computer interaction principles to design an elegant, responsive web solution that scales dynamically across hardware constraints.

02. User Research & Insights

We observed users working through various interactive flows and identified key cognitive friction points. By mapping out clear Information Architecture frameworks, we developed low-fidelity wireframes that prioritized core user tasks while reducing screen clutter.

03. The Solution & Takeaways

The finalized system implements high-contrast structural hierarchy and dynamic UI elements. By connecting systematic design layouts with intentional user interaction pathways, we created an interface that reduces cognitive loading and optimizes overall task speed.
